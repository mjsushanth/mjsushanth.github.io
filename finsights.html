FinSights: Production-Grade Financial Intelligence System

Hybrid dual-path RAG system combining structured SQL queries (DuckDB dimension tables) with semantic vector retrieval. Processes 72M→1M sentences via stratified sampling with temporal weighting across regulatory eras, achieving $0.017/query cost with no managed database overhead.
  • Data Engineering Pipeline: 30+ DuckDB SQL scripts implementing weighted multi-objective scoring, fuzzy entity matching, and conditional temporal stratification across company-year-section dimensions
  • Advanced RAG Architecture: Sentence-level embeddings with multi-query expansion, window-hopping retrieval (±3 context sentences), and citation provenance via document headers for exact source traceability
  • Evaluation Framework: Three-tier validation with deterministic gold sets (P1/P2 anchor tests achieving 82% Hit@5, P3 business-realistic suite), BERTScore 0.826 semantic alignment, and bias detection across section/year/filter regime slices
  • Production Infrastructure: AWS S3 Vectors, Bedrock LLM synthesis, Polars/Parquet logging pipeline, and serverless-ready three-tier architecture (FastAPI + Streamlit + ML orchestrator)
  • Cost Optimization: Approximated $0.05/month infrastructure, instead of expensive, managed vector databases, achieved through stratified sampling, metadata filter pushdown (50% query cost savings), and S3-native storage patterns
