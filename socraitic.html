SocrAItic Circle

Multi-agent LLM debate framework implementing structured adversarial interaction through three-phase cycles (preparation, rebuttal, evaluation) with theory-informed judge modules. Achieved measurable improvements in argument quality: 37% logical consistency, 16% factual accuracy, 19% rhetorical effectiveness, 44% strategic merit.
  • Adversarial Preparation Pipeline: Implemented invisible preparation phase enabling agents to perform evidence checking, argument mapping, and weakness anticipation before visible debate rounds, mirroring human debater research workflows with persistent memory architecture
  • Seven-Dimensional Judge Framework: Designed specialized evaluation system grounded in argumentation theory (Toulmin's Model, Pragma-Dialectical Theory, Elaboration Likelihood Model) with weighted dimensions: Logical Consistency (0.25), Factual Accuracy (0.20), Rhetorical Effectiveness (0.15), Strategic Merit (0.15), Ethical Conduct (0.10), Belief-Shift Potential (0.10), Audience Engagement (0.05)
  • YAML-Driven Orchestration: Built modular prompt management system separating debate logic from implementation, enabling rapid experimentation with different theoretical frameworks and phase-specific guidance without code modification, achieving 15-20x faster iteration cycles
  • Structured Multi-Phase Architecture: Orchestrated debate flow with preparation (strategic evidence gathering), adversarial critique mechanisms (targeted counterargument deployment), and meta-consensus evaluation aggregating diverse judge perspectives through weighted multi-dimensional optimization
  • Ollama Local Deployment: Configured local LLM infrastructure with heterogeneous model support (Mixtral, DeepSeek R1, Phi4, Llama 3.1), implementing hierarchical storage for debate histories, experiment configurations, and prompt versioning for reproducible research workflows
